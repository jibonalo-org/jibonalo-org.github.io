# জীবন আলো শেবা সংগঠন — Website

**Jiban Alo Sheba Songothon** — Official static website for a non-political, non-profit voluntary organization based in Bonpara, Natore, Rajshahi, Bangladesh.

> মানবতার সেবায়, আমরা আছি আপনার পাশে
> *In the service of humanity, we are by your side.*

---

## Project Structure

```
/
├── index.html        # Main single-page website
├── style.css         # All styles (mobile-first, responsive)
├── script.js         # Navbar, animations, counter, smooth scroll
├── images/           # Logo and image assets
│   ├── Jiban Alo Logo-V1.png     ← Organization logo (About section)
│   └── website logo-v1.png       ← Navbar & favicon logo
└── README.md         # This file
```

---

## Tech Stack

- **HTML5** — Semantic markup, ARIA accessibility
- **CSS3** — Custom properties, flexbox, grid, animations
- **Vanilla JavaScript** — No frameworks or dependencies
- **Google Fonts** — Hind Siliguri (Bangla typography)
- **Font Awesome 6** — Icons (loaded via CDN)

---

## Features

- ✅ Single-page responsive layout (mobile-first)
- ✅ Fixed sticky navbar with active link highlighting
- ✅ Hamburger menu for mobile
- ✅ Hero section with CTA buttons
- ✅ About, Mission & Vision, Activities, Why Us, Join Us, Contact sections
- ✅ Animated counter stats (Intersection Observer)
- ✅ Scroll-triggered fade-in animations
- ✅ Back-to-top floating button
- ✅ Bilingual content (Bangla + English)
- ✅ SEO meta tags + Open Graph tags
- ✅ Mailto CTA links
- ✅ Accessible (ARIA, keyboard navigable, good contrast)
